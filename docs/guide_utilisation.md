# 📚 Guide d'Utilisation - GestShop

Bienvenue dans le guide officiel de **GestShop**, votre solution complète de gestion pour votre boutique de vêtements. Ce document vous accompagnera étape par étape pour maîtriser l'outil.

---

## 🚀 1. Mise en route

### Création de compte & Connexion
1. **Accès** : Rendez-vous sur la page d'accueil de l'application.
2. **Inscription** : Si vous êtes le propriétaire, créez votre compte administrateur.
3. **Connexion** : Utilisez vos identifiants pour accéder au tableau de bord.
   * *Note : Les gérants ajoutés par l'administrateur recevront leurs accès directement.*

---

## 💻 2. Navigation Onglet par Onglet

### 🏠 Vue d'ensemble (Dashboard)
C'est votre centre de contrôle. Vous y trouverez :
*   **Indicateurs clés** : Chiffre d'affaires, nombre de ventes, produits en rupture de stock.
*   **Ventes récentes** : Un coup d'œil rapide sur les dernières transactions.

### 🛒 Point de Vente (POS) - *L'outil du vendeur*
C'est ici que vous enregistrez les ventes au quotidien.
1. **Sélection** : Cliquez sur les produits ou utilisez la barre de recherche.
2. **Panier** : Ajustez les quantités dans le panier à droite.
3. **Paiement** : Cliquez sur "Payer", choisissez le mode de règlement (Espèces, Carte, etc.).
4. **Facture** : Une fois validée, vous pouvez générer et imprimer la facture PDF pour le client.

### 👗 Produits & Catégories
*   **Ajout** : Utilisez le bouton "Ajouter un produit" pour enregistrer de nouveaux vêtements (Nom, Prix, SKU, Stock initial).
*   **Catégories** : Organisez votre boutique (ex: Chemises, Pantalons, Accessoires) pour faciliter la recherche et les rapports.
*   **Modification** : Cliquez sur l'icône bleue pour mettre à jour un prix ou une description.

### 👥 Clients
Gérez votre base de données client pour les fidéliser.
*   Enregistrez leurs coordonnées (Email, Téléphone).
*   Visualisez leur historique d'achat et le total dépensé dans votre boutique.

### 📦 Stock (Inventaire)
Suivez vos niveaux de stock en temps réel.
*   **Ajustement** : En cas de réception de marchandise ou d'erreur, ajustez manuellement les quantités.
*   **Alertes** : Les produits en stock faible sont mis en évidence pour éviter les ruptures.

### 📊 Rapports
Analysez la santé de votre business.
*   **Ventes** : Graphique d'évolution du chiffre d'affaires.
*   **Répartition** : Découvrez quelles catégories se vendent le mieux.
*   **Exports** : Téléchargez des rapports détaillés en PDF pour votre comptabilité.

### 🛡️ Utilisateurs (Gestion des Gérants)
*   Seul l'administrateur peut accéder à cet onglet.
*   Ajoutez des gérants pour votre boutique.
*   Activez ou désactivez des accès en un clic.

### ⚙️ Paramètres
Personnalisez votre expérience :
*   **Boutique** : Changez le nom de l'enseigne, l'email de contact et la devise (€, $, CFA).
*   **Profil** : Mettez à jour votre photo et vos infos personnelles.
*   **Apparence** : Basculez entre le mode **Clair**, **Sombre** ou **Système** selon vos préférences.

---

## 💡 Astuces pour une utilisation optimale

1.  **Recherche Rapide** : Utilisez la barre de recherche en haut de chaque page pour trouver instantanément un produit ou un client.
2.  **Sécurité** : Pensez à vous déconnecter si vous partagez l'ordinateur de la caisse via le bouton en bas de la barre latérale.
3.  **Facturation** : Générez toujours la facture après une vente pour garder une trace numérique propre dans l'historique des ventes.
